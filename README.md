## Product Vision

Enterprise AI ATS replicates modern enterprise hiring intelligence systems.

It combines contextual transformer-based semantic modeling, statistical NLP scoring, structured skill analytics, and executive reporting into a unified AI decision-support engine.

Designed for:

- AI-powered recruitment SaaS platforms  
- Resume optimization engines  
- HR analytics systems  
- Enterprise-grade applicant screening solutions  

</div>

## Core Intelligence Modules

### Semantic Alignment Engine
Context-aware similarity analysis using Sentence Transformers (MiniLM) to evaluate deep resume-to-job alignment beyond surface keywords.

### Statistical Relevance Engine
TF-IDF vectorization with cosine similarity to quantify keyword impact and measurable relevance.

### Skill Intelligence System
- Automated skill extraction  
- Matched vs Missing skill detection  
- Gap scoring analytics  
- Alignment percentage calculation  

### Resume Structural Analyzer
Automatic detection of:
- Professional Summary  
- Technical Skills  
- Work Experience  
- Education  
- Certifications  

### Resume Quality Index
Evaluates:
- Structural completeness  
- Keyword density  
- Content richness  
- Professional clarity  

### Executive Reporting Engine
Generates boardroom-ready PDF reports including:
- Composite ATS Score  
- Weighted performance breakdown  
- Radar analytics visualization  
- Strategic improvement recommendations

## System Architecture

```
RESUME_SCANNER/
│
├── app.py                 # Main Streamlit application entry point
├── dashboard.py           # Analytics & score visualization layer
├── ats_engine.py          # Core ATS scoring engine
├── skills.py              # Skill extraction & comparison module
├── analytics.py           # Similarity scoring & radar analytics
├── section_analyzer.py    # Resume section detection engine
├── quality_analyzer.py    # Resume quality evaluation logic
├── report_generator.py    # Executive PDF report builder
├── diagnostics.py         # Debugging & validation utilities
├── upload.py              # Resume upload handler
├── header.py              # UI header component
├── footer.py              # UI footer component
├── config.py              # Global configuration settings
│
├── requirements.txt       # Project dependencies
├── runtime.txt            # Deployment runtime specification
├── .gitignore
└── README.md
```
